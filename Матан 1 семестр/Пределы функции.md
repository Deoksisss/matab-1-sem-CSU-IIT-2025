**Опр1**
Число $A$ называется пределом функции $f(x)$  при $x \to a$ если:
1) По Гейне
$$\forall\ \{x_n\} \in \mathbb{X}\ lim_{n\to\infty}x_n = a\implies lim_{n\to\infty}f(x_n) = A$$
2) По Коши
$$\forall\ \varepsilon > 0,\exists\ \delta > 0, \forall\ x \in \mathbb{X}\ 0<|x-a| < \delta \implies |f(x) - A| < \varepsilon$$
3) По Коши через окрестности
$$\forall\ \varepsilon > 0,\exists\ \delta > 0, \forall\ x \in O_{\delta}^o\ {a}\ \implies f(x) \in O_{\varepsilon}(A)$$
## *Th*
$$lim_{x\to a}f(x) = A\ (По\ Гейне)\iff lim_{x \to a}f(x) = A \ (По\ Коши)$$
**Опр2**
Функция $f:x \to y$ называется ограниченной, если $\exists\ C \in \mathbb{R}: \forall\ x \in D(f),\ x \leq |f(x)| \leq C$
**Опр3**
Функция ограниченна снизу если $$\exists\ C \in \mathbb{R}: \forall\ x \in D(f),\ x \leq |f(x)| \leq C \iff f(x) \geq C$$
Функция ограниченна сверху если $$\exists\ C \in \mathbb{R}: \forall\ x \in D(f),\ x \leq |f(x)| \leq C \iff f(x) \leq C$$
**Опр4**
Функция ограниченна локально если $$\exists\ C\in \mathbb{R},\ \exists\ \mathbb{A}\subseteq D(f)\ \forall\ x\in \mathbb{A},\ f(x) \leq C$$
## *Th Свойства предела функции в точке*
Пусть $\exists\ \mathbb{A},\mathbb{B} \in \mathbb{R},\ f:x\to y,\ g: x \to y$
$lim_{x \to a} f(x) = A$
$lim_{x \to a} g(x) = B$
Тогда:
1) $lim_{x \to a} (f(x) + g(x)) = A+B$
2) $lim_{x \to a} c * f(x) = c*A$
3) $lim_{x \to a} (f(x)*g(x)) = A*B$
4) Если $B \neq 0$
$lim_{x \to a} \frac{f(x)}{g(x)} = \frac{A}{B}$
## *Th Предел функции и неравенства*
Пусть
$$\mathbb{E} \subseteq D(f)\ \cap\ D(g),\forall\ x \in \mathbb{E}\ f(x) \leq g(x)\ и \ a \in \mathbb{E}$$
Тогда $$lim_{x \to a} f(x) = A,\ lim_{x \to a} g(x) = B \implies A\leq B$$
Пусть $$\mathbb{E} \subseteq D(f)\ \cap\ D(g),\forall\ x \in \mathbb{E}\ f(x) \leq g(x)\leq h(x)$$
Тогда если$$lim_{x \to a} f(x) = C\ и\ lim_{x \to a} h(x) = C $$
То $$lim_{x \to a} g(x) = C$$
**Опр5**
Пусть $f:x\to y,\ g:\mathbb{Y} \to \mathbb{R}$
$x,y \in \mathbb{R}$
$\exists\  lim_{x \to a} f(x) = A$
$\exists\ lim_{y \to a} g(y) = B$
Тогда $g\circ f(x) = g(f(x))$ называется композицией $\implies$
$$lim_{x \to a} g\circ f(x) = lim_{y \to A} g(y) = B$$

$$lim_{n \to \infty} x_n = +\infty \iff x_n>\varepsilon$$

## База - основание предельного перехода
Для числовых последовательностей $n\to \infty$ - база: $\exists N\in \mathbb{N}, \forall n>\mathbb{N}$ 
Для функции 
1) $x \to x_0$ $\forall \delta > 0,\ 0<|x_n - x_0| < \delta$
2) $x \to \infty \iff \forall\ \delta\ |x| > \delta$
3) $x \to +\infty \iff \forall\ \delta > 0\ x > \delta$
4)  $x \to -\infty \iff \forall\ \delta > 0\ x < -\delta$
