$\mathrm{I}$ Аксиомы сложения
1) Нейтральные элементы $\exists \ 0 \in \mathbb{R}: \forall\  x \in \mathbb{R} \implies 0 + x = x + 0 = x$  
2) Обратный элемент $\forall\  x \in \mathbb{R},  \exists \ (-x) \in \mathbb{R} : x + (-x) = 0$
3) Коммутативность $\forall x,y \in \mathbb{R} : x + y = y + x$
4) Ассоциативность $\forall x,y,z \in \mathbb{R} : (x+y) + z = x + (y + z)$
$\mathrm{II}$ Аксиомы умножения
5) Нейтральные элементы $\exists \ 1 \in \mathbb{R}: \forall\  x \in \mathbb{R} \implies 1 * x = x * 1 = x$  
6) Обратный элемент $\forall\  x \in \mathbb{R},  \exists \ (x^-1) \in \mathbb{R} : x * (x^-1) = 1$
7) Коммутативность $\forall x,y \in \mathbb{R} : x * y = y * x$
8) Ассоциативность $\forall x,y,z \in \mathbb{R} : (x*y) * z = x * (y * z)$ 
Сложение и Умножение связаны операцией **дистрибутивности**:
$$\forall x,y,z \in \mathbb{R},\  x*(y + z) = xy + xz$$
$\mathrm{III}$ Аксиомы порядка
9) $\forall x \in \mathbb{R}, \ x\leq x$
10) $\forall x,y \in \mathbb{R} (x\leq y \land y\leq x) \implies x = y$
11) $\forall x,y,z \in \mathbb{R} (x\leq y \land y\leq z) \implies x \leq z$
12) $\forall x,y \in \mathbb{R} \implies x\leq y \lor y\leq x$  
**Замечание** 
9-11 аксиомы частичного порядка
9-12 аксиомы полного линейного упорядоченного множества
**Связь сложения и порядка** ($\mathrm{I}\ и\ \mathrm{III}$)
$$\forall x,y,z \in \mathbb{R}\ если\ x \leq y \implies x + z\leq y + z$$
**Связь умножения и порядка** ($\mathrm{II}\ и\ \mathrm{III}$)
$$\forall x,y,z \in \mathbb{R}\ если\ x > 0 \land y > 0 \implies x * y > 0$$
$\mathrm{IV}$ Аксиома полноты
Пусть $X,Y \subset \mathbb{R}$
$$\forall x \in X, y \in Y,\ где\ x \leq y \implies \exists\ c \in \mathbb{R}: x \leq c \leq y $$
